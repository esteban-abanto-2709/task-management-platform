# Task Management Platform

A full-stack task management platform built with NestJS and Next.js, focused on clean architecture, real-world backend patterns, and maintainable frontend structure.

## Tech Stack

### Backend

- NestJS
- TypeScript
- Prisma ORM
- PostgreSQL
- JWT Authentication
- class-validator / class-transformer

### Frontend

- Next.js (App Router)
- TypeScript
- TailwindCSS

## Features (MVP)

- User authentication (JWT)
- Project management
- Task management
- Task status workflow
- Protected API routes

## Project Structure

- `apps/api`: Backend API (NestJS)
- `apps/web`: Frontend application (Next.js)
